## 🚀 快速开始

### 在线使用
直接访问：[https://index.jimmyxuexue.top](https://index.jimmyxuexue.top)

### 浏览器插件安装
1. 前往 [Release 页面](https://github.com/Jimmylxue/snowIndex/releases) 下载最新版本
2. 解压下载的文件
3. 在浏览器扩展管理页面启用"开发者模式"
4. 点击"加载已解压的扩展程序"，选择解压后的文件夹

### 基础指令示例
```bash
baidu 前端开发          # 使用百度搜索
