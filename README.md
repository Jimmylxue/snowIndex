## 关于 SnowIndex

就是一个将**大而全的功能全部继承在一个小而美的网页**上的一个浏览器主页，可不断继续开发，完善和添加更多的功能！

### 快速上手

打开网站：[传送门](http://www.jimmyxuexue.top:668)

```
baidu 前端开发
github snowIndex
按键 Ctrl + L 清屏
help => 查看帮助文档
```

目前已支持 30+指令：
